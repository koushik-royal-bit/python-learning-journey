# Day 010 - Python Lists and Nested Lists Practice

## Overview
This project is a hands-on Python practice exercise focused on working with lists and nested lists. It demonstrates how to create, access, update, and modify list data using common list methods such as `append()`, `insert()`, `remove()`, and `extend()`.

## What I Learned
- Creating lists in Python
- Working with nested lists
- Accessing elements using indexes
- Updating values inside nested lists
- Adding items with `append()` and `insert()`
- Removing items with `remove()`
- Expanding a list with `extend()`

## Project Example
The practice script uses a nested list of skills and updates it step by step to show how lists can be modified dynamically.

## Sample Output
```python
Original Skills:
[['Python', 95], ['SQL', 80], ['Git', 70]]

Top Skill:
Python

Git Score:
70

Updated Skills:
[['Python', 95], ['Pandas', 85], ['SQL', 90], ['Linux', 75], ['Docker', 65], ['AWS', 60]]

Final Skill List:
[['Python', 95], ['Pandas', 85], ['SQL', 90], ['Linux', 75], ['Docker', 65], ['AWS', 60]]
```

## Learning Notes
- Lists are one of the most useful built-in data structures in Python.
- Nested lists allow you to store related data in a structured way.
- Indexing is essential for accessing specific values inside lists and nested lists.
- Methods like `append()` and `extend()` are helpful when growing a list dynamically.
- Practice with small examples helps build confidence in understanding list behavior.
